# RelaxSlot

Play Slot Machine Game
Link: https://relax-slot.web.app/

Pixi Slot Machine is a bare minimum slot machine game to get familiar with Pixi.js. 

Game parameters are configurable through a configuration file.

![Gameplay](https://i.imgur.com/6SjKO0w.gif)

## Installation

```
$ git clone https://github.com/petarstajic/RelaxSlot.git
$ cd RelaxSlot
$ npm i
```

## Running

```
$ npm run start
```

## Build

```
$ npm run build
```
