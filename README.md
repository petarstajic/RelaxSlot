# RelaxSlot

Play Relax Slot Machine Game

Link: https://relax-slot.web.app/

This game is a bare minimum slot machine game made as an interview test task and to get familiar with PixiJS.

The main technologies are PixiJS and VanillaJS.

It is my first contact with PixiJS. Basically, it's a Frankenstein made out of a bunch of Pixi examples that are available online. I'm not proud of my creation, but it works. :smirk:

As I was very limited on time, there is a lot of things that could be better, including general code structure, more assets, and advanced animations.

![Gameplay](https://i.imgur.com/6SjKO0w.gif)

## Installation

```
$ git clone https://github.com/petarstajic/RelaxSlot.git
$ cd RelaxSlot
$ npm i
```

## Running

```
$ npm run start
```

## Build

```
$ npm run build
```
